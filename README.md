## Desafios de Projeto

### Otimizando Sistema Bancário Com Funções Python
Além do sistema proposto, esse sistema é composto por 2 partes: admin e usuário. Onde p admin tem acesso a todas as funções, enquanto o usuário é restrito. como descriminado abaixo.
<br/>
Admin:
* Criar Usuário
* Criar Conta Corrente
* Depositar
* Sacar
* Extrato
* Listar Conta
<br/>

Usuário:
* Depositar
* Sacar
* Extrato
<br/>
No início do sistema é pedido uma autenticação, onde o CPF é usado como senha para dar permissão de admin ou usuário.<br/>
O CPF reservado para o admin é: 12345678909<br/>
O usuário só conseguirá acesso após o gerente, admin, cadastrar seus dados no sistema e criar sua conta corrente.<br/>
Se o usuário tentar acessar o sistema sem cadastro, ele será informado que deverá procurar o gerente para a realização do cadastro.
